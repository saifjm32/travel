Travel Buddy — AI-Powered Travel Planner

Travel Buddy is an AI-powered web application that generates personalized travel plans based on user destination and preferences.
The system creates a custom user avatar and displays the itinerary on an interactive map with 360° views of recommended locations.

Features

AI-generated travel itineraries

Personalized user avatar

Smart destination recommendations

Interactive map visualization

360° panoramic place views

How It Works

User selects destination and preferences

AI API generates a customized travel plan

Avatar is created for personalization

Results are shown on an interactive map with 360° views

Technologies

Frontend: HTML, CSS, JavaScript

AI: External AI APIs (itinerary generation)

Maps & Visualization: Map API + 360° Street View

API Keys

API keys are not included in this repository.
Users must provide their own keys via environment variables or configuration files.

Project Type

AI-assisted web application focused on personalization, system design, and interactive visualization.
